# easyChat 增强版修改说明

本项目基于原 easyChat 项目二次开发，新增了以下功能：

## 新增功能
### 1. 智能随机定时生成
- 根据待发送内容数量自动生成对应数量的定时任务
- 时间平均分布在指定时间段内，支持随机波动范围
- 每个时间点发送对应序号的内容，发完即止
- 无需手动设置发送范围，默认按顺序发送全部内容

### 2. 一键清空所有定时
- 新增"清空所有定时"按钮，一键删除所有定时任务
- 有二次确认提示，防止误删

### 3. 发送后自动删除文件
- 可选开启，文件发送成功后自动删除源文件
- 配置自动保存，重启程序保持上次设置

### 4. 文件夹同步功能
- 自动同步指定文件夹的内容到待发送列表
- 支持自动同步和手动同步两种方式
- 不会删除手动添加的消息和非同步文件夹的文件
- 同步时保留选中状态和滚动位置，不影响正常操作

### 5. 进程完全退出修复
- 修复关闭窗口后后台进程残留的问题
- 关闭程序时自动停止所有定时器和线程，释放资源

### 6. 兼容性优化
- 支持全兼容打包，包含所有运行时依赖
- 提供便携版打包，方便在其他电脑上运行

## 兼容性
- 支持微信 4.1 版本
- 兼容 Windows 10/11 系统
- 提供便携版，无需安装Python环境即可运行
